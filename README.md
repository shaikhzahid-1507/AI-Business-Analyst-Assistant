
# 🚀 AI Business Analyst Assistant

## Enterprise Customer Analytics Dashboard powered by AI

Analyze customer data, generate executive insights, detect risks, identify growth opportunities, and make smarter business decisions.

---

## 🌐 Live Demo

https://ai-business-analyst-assistant-dashoxdervuywwgybt4w5n.streamlit.app/

---

## 📌 Overview

AI Business Analyst Assistant is a professional business analytics platform built with Python and Streamlit. It enables users to upload CSV/Excel datasets, explore KPIs, perform profiling, generate AI-powered insights, identify risks, and export executive reports.

---

## ✨ Features

- CSV & Excel Upload
- Data Profiling
- KPI Dashboard
- Executive Dashboard
- AI Business Insights
- Executive Summary
- Recommendations
- Risk Assessment
- Growth Opportunities
- PDF Report Export
- Responsive UI
- Error Handling
- Caching

## ⚙ Installation

```bash
git clone https://github.com/shaikhzahid-1507/AI-Business-Analyst-Assistant.git

cd AI-Business-Analyst-Assistant

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

streamlit run app/app.py
```
